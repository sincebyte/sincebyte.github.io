1. 特点概述

Emacs 是一个非常古老且强大的文本编辑器，它最初由理查德·斯托曼（Richard Stallman）开发，旨在提供一个高度可定制的编辑环境。Emacs 的特点包括内置的Lisp解释器，允许用户通过编写Lisp代码来扩展和定制编辑器的功能。然而，对于一些用户来说，Emacs 的学习曲线可能有些陡峭。

Doom Emacs 是一个构建在Emacs之上的配置框架，旨在简化Emacs的配置过程并提供现代化的用户体验。它整合了许多流行的Emacs插件和工具，并提供了一个优雅的默认配置，使得用户可以快速上手并享受Emacs的强大功能。Doom Emacs 的历史可以追溯到对Emacs社区的贡献和对现代化工具的需求。

Neo emacs 是一个 GNU Emacs 的配置框架，它基于 doom emacs，本站专注于 web 应用程序编码环境，但这仅只是 Emacs 众多能力面中的很小部分，也丝毫不影响你以此作为基础向其他专业方向扩充。另外本站提供 Neo Emacs 的安装器，您可以通过安装器来便捷式的安装 Neo Emacs，跳过远程依赖的同步下载和编译及配置步骤，开箱即用。

Neo emacs 具有以下特点：

  • 输入法 ：适配 rime 配置，默认支持小鹤音形、小鹤双拼、全拼、自然码双拼。
  • 翻译插件 ：go-translate ，学习英语没得说了，支持国内外主流翻译 API。
  • Evil ：这是键盘流的基础，如果你曾经用过vim，oh 除非您是另外一个流派。
  • Java ：支持 maven 和 gradle 项目，用了3年了，请注意 eclipse 要开始超车了。
  • Golang ：所幸 golang 官方提供语言服务，Java 什么时候也官方出个？期待一下
  • Python ：由插件 pyright 提供，python 有那么多库太有吸引力了。遥遥领先~
  • Vue ：由插件 web-mode 提供，语言服务由 vls 提供。本站Vue2.0写的😂😂。
  • Libvterm ：终端模拟器, vterm 支持 fish、zsh、bash，可惜 windows 下不支持
  • Eredis ：redis 客户端，注重性能和稳健性。我想这个插件应该需要再扩展扩展…
  • Magit ：Magit 的界面设计简洁直观，让 Git 的操作高效和便捷。emacs 独家享有
  • Ejc-sql ：将 Emacs 变成一个简单的 SQL 客户端，支持各种数据库。Mysql、Oracle、clickhouse、Sqlserver…
  • Org-roame ：知识管理系统 借鉴了 GTD 方法的原理，为非分层笔记、双链笔记提供了解决方案。可谓 emacs 杀手级应用。
  • 作图软件 ：plantUML、D2、dotgraiz、gnuplot，如果您正在出书或者需要写开发文档，代码作图应该值得您体验。
  • Rest-client ：http客户端工具，像 Postman 一样手动探索和测试 HTTP REST Web 服务的工具。可以编程式的那种。

本站前端、后端、数据库、测试全部由 neoemacs 开发， 同时您现在看到的这篇文档也是由 ｢ org mode ｣ 编写完成的。
